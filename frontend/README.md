# SheepParter Frontend

React + Vite + TypeScript monorepo（单体多包仓库） using npm workspaces（工作区）.

## Apps

- `apps/user-web`：用户 Web 端。
- `apps/admin-web`：管理员 Web 端。

## Packages

- `packages/ui`：共享 UI（用户界面）组件。
- `packages/api-client`：共享 API client（接口客户端）。

## Commands

```powershell
npm install
npm run dev:user
npm run dev:admin
npm run typecheck
npm run build
```
